## ✨ Overview

The **M Chick Fried Chicken Website** is a modern restaurant website built to provide an attractive online presence for M Chick. Customers can browse the full menu, explore signature dishes, discover current offers, and place orders with a single click through **WhatsApp** or **Phone Call**.

The website is fully responsive and optimized for desktop, tablet, and mobile devices.
